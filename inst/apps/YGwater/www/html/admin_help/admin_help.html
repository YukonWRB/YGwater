YGwater admin help

This placeholder help file is for the admin side of the YGwater application.

Admin workflows

Use the admin navigation menus to manage monitoring locations, continuous data, discrete data, files, field visits, equipment, boreholes and wells, reference data, acquisition and telemetry metadata, users, notifications, news content, and feedback.

Page-specific help

The Current page help menu item opens help for the last active admin page. If a dedicated help file has not been written yet, YGwater opens a generic placeholder for that page.

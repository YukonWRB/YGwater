Add discrete data

Use this page to add discrete sample results to AquaCache. The workflow creates sample records in discrete.samples and result records in discrete.results from either uploaded files or manually entered rows.

Location and sub-location

Select the location before uploading or entering results. Select the sub-location when results belong to a specific point within the location. If the needed location or sub-location does not exist, the page can send you to the relevant location tool to create it first.

Input methods

Use file upload when you have a spreadsheet or tabular export. Use manual entry for a small number of results. Uploaded files can be mapped from wide or long formats, depending on how parameters, units, dates, and values are arranged.

Mappings

Mappings tell the app where to find parameters, units, dates, values, and samples in the uploaded file. Save reusable mappings for common lab exports or repeated field templates. Before saving a mapping, verify that parameter columns resolve to the intended AquaCache parameters.

Parameters, units, and values

Check parameter names and units before upload. A result should only be saved when the parameter is identified and the value is valid. Avoid uploading ambiguous columns until the mapping is corrected.

Documents

Attach supporting documents when they should be linked with the uploaded samples, such as lab reports, field sheets, or chain-of-custody documents.

Before uploading

Review the preview table carefully. Confirm sample dates, location, sub-location, parameter matches, units, values, and attached documents. Upload only after the preview reflects the records that should be added to AquaCache.

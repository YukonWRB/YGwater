Manage users

Use this page to create, modify, and review AquaCache users and user groups when your account has role-management privileges. You'll find a drop-down menu at the top-left of the page to switch between various tasks.

Creating new groups and users

In PostgreSQL, the only difference between a user and a group is that a user can log in, while a group cannot. Groups are used to manage sets of privileges that can be assigned to multiple users.

To create a new group, select the Create new group option from the drop-down menu. Enter the group name and click Create group. IMPORTANT: Use the naming convention 'orgName_group' (e.g. 'yg_editor_group')so that we can more easily identify groups from users.

Privileges

Review inherited privileges before changing direct grants. Role and table privilege changes affect database access outside the Shiny application as well.

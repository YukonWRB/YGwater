Add / edit samples

Use this page to create or update records in discrete.samples. A sample record describes a sampling event: where and when the sample was collected, the sample type, collection method, owner, access groups, and related field details. Analytical results are managed separately in the discrete-data results workflow.

Add or modify mode

Choose Add new to create a sample. Choose Modify existing to select samples from the table and update their metadata. Multi-sample edit can update selected fields across several samples at once; fields such as location, sub-location, elevation/depth, sample datetime, target datetime, and import source ID remain single-sample edits.

Location and time

Every sample needs a location and datetime. Select a sub-location when the sample belongs to a specific point within the parent location. Use the timezone controls carefully so the saved datetime represents the true sampling time.

Sample details

Select media, collection method, and sample type. Use volume, purge volume, purge time, flow rate, and wave height when those values are relevant to the sampling method. Linked sample can be used when samples should be related to one another.

Grades, approvals, and qualifiers

Sample grade, approval, and qualifier fields describe sample-level quality and review status. Use them to document whether a sample is provisional, approved, rejected, qualified, or otherwise limited before results are interpreted.

Organizations and documents

Set the owner and, when available, contributor, commissioning organization, and sampling organization. Attach documents when supporting field sheets, chain-of-custody forms, photos, or other files are relevant to the sample record.

Access and update locking

The Share with groups field controls which roles can see the sample. public_reader makes the sample broadly visible. Use Lock sample from updates when a sample should not be overwritten by later imports or synchronization processes.

Before saving

Confirm location, sub-location, datetime, media, collection method, sample type, owner, access groups, and any quality fields. After creating sample records, add or import analytical results in Add discrete data.
